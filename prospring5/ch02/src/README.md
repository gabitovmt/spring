# Построение примера приложения "Hello World"

Рассматривать приложения в следующем порядке.

- `HelloWorld`;
- `HelloWorldWithCommandLine`, получение сообщения из аргументов командной строки;
- `HelloWorldDecoupled`, логика воспроизведения и получения сообщений вынесена в отдельные компоненты;
- `HelloWorldDecoupledWithFactory`, фабричный класс для извлечения имён классов реализации и получения их экземпляров
  по мере необходимости.

# Реорганизация кода средствами Spring

Недостатки предыдущих примеров.
1. Много связующего кода для соединения всех частей в единое приложение.
2. Должны вручную предоставлять реализацию интерфейса `MessageRenderer` вместе с экземпляром реализации
   интерфейса `MessageProvider`.
